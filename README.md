# MyBlogApp

Welcome to MyBlogApp, a powerful and versatile blogging platform where you can unleash your creativity and share your thoughts with the world!

![MyBlogApp Screenshot](screenshot.png)

## Features

- **Create and Publish Articles:** Write and publish your own articles effortlessly.
- **Rich Text Editor:** Enjoy a user-friendly rich text editor for composing your posts with ease.
- **Tagging and Categorization:** Organize your articles with tags and categories for easy navigation.
- **User Authentication:** Securely sign up, log in, and manage your profile.
- **Social Sharing:** Share your articles on social media platforms to reach a wider audience.
- **Commenting System:** Engage with your readers through comments on your articles.
- **Responsive Design:** Access and use the platform seamlessly on any device, whether desktop, tablet, or mobile.

## Getting Started

To get started with MyBlogApp, follow these simple steps:

### Prerequisites

- Node.js and npm installed. You can download and install Node.js from [here](https://nodejs.org/).

### Installation

1. Clone the repository:

```sh
    git clone https://github.com/CodexEmmzy/MyBlogApp.git
```
2. Navigate to the project directory:
```sh
cd MyBlogApp
```

3. Install dependencies:

```sh
npm install
```

4. Start the development server:

```bash
    npm run dev
    # or
    yarn dev
    # or
    pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
